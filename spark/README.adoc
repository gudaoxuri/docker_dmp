== Spark Environment

*Based tairangroup/hadoop:latest , Depend gudaoxuri/scala_2.11:latest*

=== Features

* SSH server & client ( Port: 22 User: root Password: 123456 )
* Hadoop 2.7.x
* Spark 2.3.x

=== Examples

Start Spark:

 docker run --name spark -h spark -P -e KEEP=true -d gudaoxuri/spark
